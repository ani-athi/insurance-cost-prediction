# 🏥 Insurance Cost Prediction using Linear Regression

## 📌 Overview

This project predicts medical insurance charges based on personal and financial attributes such as BMI, salary, and past consultations.

## 🧠 Model Used

* Linear Regression

## ⚙️ Workflow

1. Data cleaning (handling missing values)
2. Outlier removal
3. Label encoding of categorical variables
4. Train-test split (80-20)
5. Model training
6. Prediction and evaluation

## 📊 Evaluation Metric

* R² Score (coefficient of determination)

## 📈 Result

The model successfully captures the relationship between input features and insurance charges.

## 🚀 How to Run

* Open the notebook in Google Colab
* Run all cells
